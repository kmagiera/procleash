Opis szczegółowy

Reprezentuje argumenty. Argumenty moga byc dlugie (takie po podwujnej kresce) lub krotkie (takie po pojedynczej kresce) Pola: argtype - OT_LONG dla argumentow dlugich, OT_SHORT dla krotkich argument - dla krotkich jest tam pojedynczy znak, dla dlugich lancuch znakow event - flaga ze zdarzeniami (zdarzenia z option_event) revent - pole z wynikami przetowrzenia (opcjie z option_revent) value - wartosc parametru
